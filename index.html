Comandos de Decisão

Condições - Expressões Relacionais

Uma condição na linguagem Python é definida como uma expressão que pode ser verdadeira ou falsa. A este tipo de expressão dá-se o nome de expressão relacional ou comparação. Por exemplo, 3 > 2 uma expressão relacional que possui valor verdadeiro. Por outro lado, 4 < 1 é uma expressão que possui valor falso. Os operadores (< e >) usados nos exemplos acima são chamados de operadores relacionais pois possibilitam saber qual a relação existente entre seus dois operandos. Além destes dois existem mais 4 operadores relacionais, que podem ser vistos na tabela seguir:

Tabela de Operadores Relacionais

Tabela - Operadores Relacionais

Operador Descrição
== Igual a
!= Diferente
>= Maior ou igual
> Maior que
< Menor que
<= Menor ou igual

Comando IF

O comando if serve para selecionar que parte de um código será executado.

Exemplo de If:
print("Digite um número:") a = int(input()) if (a % 2 == 0) and (a < 100): print("O número é par e menor do que 100") if (a % 2 == 0) and (a >= 100): print("O número é par e maior ou igual que 100") if (a % 2 != 0) and (a < 100): print("O número é ímpar e menor do que 100") if (a % 2 != 0) and (a >= 100): print("O número é ímpar e maior ou igual que 100")

Comando Else

Para evitar uma sequência longa de IFs é possível agregar o comando else a um comando if. O trecho apresentado a seguir classifica uma pessoa de acordo com a regra descrita na tabela.

Tabela de Categorias por Idade

Idade Categoria
0-1 Recém Nascido
2-12 Criança
13-18 Adolescente
19-60 Adulto
60-80 Idoso
>80 Longevo
Exemplo de Else:
a = input("Digite a idade da pessoa: ") idade = int (a) if idade <=1: print ("Bebê") else: if idade < 13: print ("Criança") else: if idade < 17: print ("Adolescente") else: if idade < 60: print ("Adulto") else: if idade < 80: print ("Idoso") else: print ("Longevo")

Comando Elif

Quando for necessário fazer um novo teste após um ELSE, como no exemplo anterior, é possível simplificar o comando usando um ELIF.

Exemplo de Elif:
valor = input("Digite um valor: ") if isinstance(valor, int): print("É um inteiro") elif isinstance(valor, float): print("É um float") elif isinstance(valor, str): print("É uma string") else: print("Tipo de dado desconhecido")
